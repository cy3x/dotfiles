Setup:

- [terminus](http://terminus-font.sourceforge.net) - font
- deep-space - colorscheme
- lf - file manager

Usage:
------

    cd ~
    git clone https://github.com/cy3x/dotfiles
    cd dotfiles
    stow polybar (or i3, rofi, dmenu, etc.)
